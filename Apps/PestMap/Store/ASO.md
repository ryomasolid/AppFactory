## 説明文（最大4000字）
```
害虫対策、「いつ・どこに置いたか」を忘れない。

PestMap は、ブラックキャップなどの害虫対策グッズを「家のどこに設置したか」を間取り図上に記録し、次の交換・対策のタイミングを通知でリマインドするアプリです。

■ こんな人に
・ブラックキャップ等を置いたけど、場所も時期も忘れがち
・定期的な対策（毎月/数ヶ月ごと）を続けたい
・家のどこに何を設置したか家族と把握したい

■ 主な機能
・間取り図を写真で取り込み、または白紙の方眼から作成
・設置・対策場所をマーカーで記録（種別・メモ・写真）
・次回の予定を通知でリマインド（毎月/3ヶ月/半年/毎年の繰り返しも）
・「実施した」を記録すると次回を自動で再設定（履歴も残る）
・予約中のリマインドを一覧で確認、期限切れも一目で

■ 安心・プライバシー
・あなたの写真・メモ・予定は端末内に保存され、外部に送信されません
・アカウント登録は不要です

※ 通知の利用には許可が必要です。
※ 無料版では広告（Google AdMob）を表示します。
```

## スクリーンショット構成（6.7" と 6.5"/5.5"）
1. フック: 「害虫対策、ちゃんと続く」＋ 間取り＋マーカーのイメージ
2. 間取り図にマーカー配置（種別の色分け）
3. マーカー編集（種別／メモ／設置場所の写真／次回予定）
4. リマインド一覧（期限切れ強調）
5. 繰り返し設定＋通知のイメージ
6. 「完全オンデバイス／外部送信なし」の安心訴求
